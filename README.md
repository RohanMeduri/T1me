# T1me

## Inspiration
Our inspiration was the theme for this project, to elevate the ordinary. We decided to take a common idea and add our own twist.
## What it does
This website is a user friendly time management and focus booster, which improves the efficiency of students.
## How we built it
We built the website using Html, CSS, and JavaScript.
## Challenges we ran into
We did not know the direction our project would go, and the features that we could have implemented.
## Accomplishments that we're proud of
We are particularly proud with the user friendly design, which makes the UI clean and refreshing for the eyes.
## What we learned
We learned time management, which is why we believe this project is very useful for students.
## What's next for T1me
We plan to improve the design and thought behind the project, through adding features and study techniques.
